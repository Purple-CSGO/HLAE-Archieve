# HLAE-Archieve
该仓库存放[HLAE](github.com/advancedfx/advancedfx)的安装zip包和exe安装器，用于部分地区的加速下载
> This repo automatically stores hlae's release assets/files when hlae has a new release to archieve and enable Jsdelivr
